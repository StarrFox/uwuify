# uwuify

Command line uwuification

# Installation
```shell
# Stable
pip install -U uwuify

# Dev
pip install -U git+https://starrfox/uwuify@master
```

# Usage
```shell
echo hello | uwuify -> hewwo
uwuify --input file.txt --output output.txt -> contents of file.txt uwuifed into output.txt
```
